# WTApp - Real-time Translation for WhatsApp

Sitio web de presentación para WTApp, una aplicación Android que integra traducción en tiempo real directamente en WhatsApp.

## Estructura del Proyecto

```
WTApp_CoreNest/
├── index.html          # Página principal del sitio
├── style.css           # Estilos del sitio
└── imagenes/           # Recursos gráficos
    ├── wtapp_logo.png
    ├── traducirchat1.png
    ├── traducirchat2.png
    ├── calltranslate1.png
    ├── calltranslate2.png
    └── wtappanuncio.jpg
```

## Cómo usar

1. Abre `WTApp_CoreNest/index.html` en tu navegador
2. El sitio incluye un selector de idioma (EN/ES) en la esquina superior derecha
3. Todas las imágenes y recursos están en la carpeta `imagenes/`

## Características

- Sitio bilingüe (Inglés/Español)
- Diseño responsive
- Secciones: Historia, Problema, Solución, Mercado, Tecnología, Modelo de Negocio, y más
- Galería de imágenes demostrativas
